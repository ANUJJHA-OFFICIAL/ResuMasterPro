# 🚀 ResuMasterPro

**ResuMasterPro** is a user-friendly web application designed to help users create professional resumes effortlessly — completely free of cost.

It is built for:

- 🎓 Students
- 💼 Working Professionals
- 🧑‍💻 Job Seekers
- 🌍 Anyone who wants a clean and modern resume

---

## ✨ Features

- 🧾 Easy resume creation with a simple interface
- ⚡ Fast and responsive design
- 🎨 Clean and modern resume templates
- 🆓 Completely free to use
- 🌐 Accessible from anywhere via browser

---

## 🛠️ Tech Stack

- React + Vite
- HTML5 + CSS3
- JavaScript / TypeScript

## 💻 Run Locally

### Prerequisites:

- Node.js installed

### Steps:

1. Clone the repository:

```bash
git clone https://github.com/ANUJJHA-OFFICIAL/ResuMasterPro.git
```

2. Navigate to the project folder:

```bash
cd ResuMasterPro
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

---

## 🚀 Deploy

```bash
npm run deploy
```
